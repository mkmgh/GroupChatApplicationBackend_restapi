Events to listen

verifyUser event
This event ("verifyUser") has to be listened on the user's end to verify user authentication.
user will only be set as online user after verification of authentication token.
online-user-list event
This event ("online-user-list") has to be listened on the user's end to identify all available users that are currently online.
The output will be an object, object has key as userId and value as full name.
message event
This event ("message") has to be listened to get all chats on chatRoom.The chat which is sent by user will
be listened to each member who has joined a room.
Example output it as follows:
                            {
                                chatId: 'unique chat id',
                                message: 'you message',
                                createdOn: date, 
                                receiverId: 'chatRoomId',
                                receiverName: 'chatRoomName',
                                senderId: 'userId of sender',
                                senderName: 'full name of sender'
                            }
typing event
This event ("typing") has to be listened to get data who is typing.This data will be provided to every
member of Room.
ClosedGroup event
This event ("ClosedGroup") has to be listened to get know if group joined by user is closed.
This event is listened by every member of Room. The group name will provided as data.
GroupCreated event
This event ("GroupCreated") has to be listened to get know if any new group has been created.
This event is listened by every user of Group Chat Application. The group name will provided as data.
GroupRemoved event
This event ("GroupRemoved") has to be listened to get know if any group has been deleted.
This event is listened by every user of Group Chat Application. The group name will provided as data.
auth-error event
This event ("auth-error") has to be listened to get know if any error has occurred on socket.
This event is emitted to every user of Group Chat Application.
disconnection
This event ("disconnection") has to be emitted, when the user gets disconnected(logout).

Events to emit

set-user event
This event ("set-user") has to be emitted when a user comes online.
User can only be set as online only after verification of authentication token. Which you have pass here. The following data has to be emitted:
authentication token
chat-msg event
This event ("chat-msg") has to be emitted to savechat message in DB and notify user in chat room
The following data has to be emitted:
                            {
                                senderName: 'John Doe',
                                senderId: 'userId of sender',
                                receiverName: chatRoomName',
                                receiverId: 'chatRoomId of Room',
                                message: 'your chat message',
                                createdOn: date
                            }
userTyping
This event ("userTyping") has to be emitted to notify every other member of chat room .
The following data has to be emitted
                           {
                                userName:'Name of User who is typing',
                                room:'ChatRoomId'
			               }
GroupClosed
This event ("GroupClosed") has to be emitted to notify every other user of Group Chat Application .
The group name has to be emitted.
newGroup
This event ("newGroup") has to be emitted to notify every other user of Group Chat Application .
The group name has to be emitted.
GrpDeleted
This event ("GrpDeleted") has to be emitted to notify every other user of Group Chat Application .
The group name has to be emitted.
userId
This event ("userId") has to be emitted to join the group via group Invite or from other groups list .
The following data has to be emitted
                            {
                                userEmail: "Email Id of the user who wants to join the group",
                                roomId : "roomId of the group which is to be joined ",
                                userId : "userId of the user who will join the chat Group"
                            }
